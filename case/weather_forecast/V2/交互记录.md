### 任务目标
实现一个轻量级的全球短期天气预报模型，采用清晰的模块化结构，便于后续进行结构改进和模块替换实验。模型输入当前时刻的全球气象场（包含地表变量和多层大气变量），输出下一时刻的预报结果。

### 输入定义

#### 地表变量
- **形状**：`(Batch, SurfaceVars, Height, Width)`
- **典型变量**：地表气压、海平面气压、2米温度、2米湿度、10米风速、10米风向等
- **推荐变量数**：7
- **分辨率**：`(721, 1440)`（纬度×经度）

#### 多层大气变量
- **形状**：`(Batch, UpperAirVars, PressureLevels, Height, Width)`
- **典型变量**：位势高度、温度、湿度、u风、v风等
- **推荐变量数**：5
- **推荐气压层数**：13
- **分辨率**：`(721, 1440)`（纬度×经度）

### 输出定义

#### 地表变量
- **形状**：`(Batch, SurfaceVars, Height, Width)`
- **变量**：与输入相同

#### 多层大气变量
- **形状**：`(Batch, UpperAirVars, PressureLevels, Height, Width)`
- **变量**：与输入相同

### 变量组织方式

#### 数据结构
- 采用分离的地表和高空分支处理
- 统一使用 `Batch`、`PressureLevels`、`Height`、`Width` 命名
- 地表分支：2D 处理，形状为 `(Batch, SurfaceVars, Height, Width)`

- **不组织成统一的 3D token 主干**，而是保持两个独立分支，各自使用对应的 PanguFuser

### 主干结构建议

#### 整体架构
采用**分离的双分支编码器-解码器架构**：

1. **输入层**：接收地表和高空气象场
2. **Embedding 层**：
   - 地表分支：`OneEmbedding(style="PanguEmbedding", ...)` → 2D patch embedding
   - 高空分支：`OneEmbedding(style="PanguEmbedding", ...)` → 3D patch embedding
3. **编码层**：
   - 地表分支：`OneFuser(style="PanguFuser", ...)` → 2D token 融合
   - 高空分支：`OneFuser(style="PanguFuser", ...)` → 3D token 融合
   - 下采样：`OneSample(style="PanguDownSample", ...)` → 降低空间分辨率
4. **中间层**：
   - 地表分支：`OneFuser(style="PanguFuser", ...)` → 2D token 融合（更高维度）
   - 高空分支：`OneFuser(style="PanguFuser", ...)` → 3D token 融合（更高维度）
5. **解码层**：
   - 上采样：`OneSample(style="PanguUpSample", ...)` → 恢复空间分辨率
   - 地表分支：`OneFuser(style="PanguFuser", ...)` → 2D token 融合
   - 高空分支：`OneFuser(style="PanguFuser", ...)` → 3D token 融合
6. **输出层**：
   - 地表分支：`OneRecovery(style="PanguPatchRecovery", ...)` → 恢复地表变量
   - 高空分支：`OneRecovery(style="PanguPatchRecovery", ...)` → 恢复高空变量

#### 关键设计决策
- **不组织成统一的 3D token 主干**：根据 Pangu 系列的设计，地表和高空分支保持独立，各自使用对应的 2D/3D 处理
