# 全球+区域混合FourCastNet实现方案

## 实现方案概述

我已经为您实现了一个全球+区域混合的天气预报系统，该系统能够融合FourCastNet官方开源权重的全球特征和局部区域的气象数据，进行高分辨率预测。

## 核心实现

### 1. 模型结构设计

创建了 `GlobalRegionHybridAFNONet` 模型，该模型包含以下核心组件：

- **GlobalFeatureExtractor**：从预训练的FourCastNet模型中提取最后一层特征
- **RegionFeatureFusion**：融合全球特征和局部区域特征
- **局部特征提取**：使用前半部分的AFNO blocks提取局部区域特征
- **特征处理**：使用后半部分的AFNO blocks处理融合后的特征

### 2. 数据加载器

创建了 `GlobalRegionERA5Datapipe` 和 `GlobalRegionERADataset`，用于加载：
- 局部区域数据
- 全球数据
- 区域信息（经纬度范围）

### 3. 训练和推理脚本

- **train.py**：用于训练全球+区域混合模型
- **inference.py**：用于加载训练好的模型并进行推理

### 4. 配置文件和文档

- **conf/config.yaml**：模型和数据配置
- **README.md**：项目说明和使用方法

## 实现细节

### 全球特征提取

使用预训练的FourCastNet模型提取全球特征，具体是提取模型的 `forward_features` 输出，即最后一层的特征。

### 区域特征融合

通过区域掩码选择与局部区域对应的全球特征，然后通过线性投影将全球特征映射到与局部特征相同的维度，最后通过AFNO block融合特征。

### 区域信息处理

区域信息包含经纬度范围 (lat_min, lat_max, lon_min, lon_max)，用于生成区域掩码，选择与局部区域对应的全球特征。

## RegionFeatureFusion中的Block

`Block`是AFNO（Adaptive Fourier Neural Operator）模型中的**基本构建块**，它是一个完整的特征处理单元。

### Block的结构组成

从代码中可以看到，Block包含以下几个核心组件：

1. **第一层归一化**：`self.norm1` - LayerNorm归一化
2. **AFNO2D滤波器**：`self.filter` - 在频域进行空间特征混合
3. **MLP层**：`self.mlp` - 多层感知机进行特征变换
4. **第二层归一化**：`self.norm2` - LayerNorm归一化
5. **跳连接**：`double_skip` - 双跳连接机制

### Block的工作原理

```python
def forward(self, x):
    residual = x
    x = self.norm1(x)           # 第一层归一化
    x = self.filter(x)          # AFNO2D频域滤波

    if self.double_skip:        # 双跳连接
        x = x + residual
        residual = x

    x = self.norm2(x)           # 第二层归一化
    x = self.mlp(x)             # MLP特征变换
    x = self.drop_path(x)       # Dropout路径
    x = x + residual            # 跳连接
    return x
```

### 在RegionFeatureFusion中的作用

在`RegionFeatureFusion`中，Block被用作**特征融合模块**：

```python
self.fusion_block = Block(
    dim=embed_dim,
    mlp_ratio=4.,
    num_blocks=num_blocks,
    sparsity_threshold=0.01,
    hard_thresholding_fraction=1.0
)
```

它的作用是：
1. **融合局部特征和全球特征**：接收融合后的特征
2. **频域特征混合**：通过AFNO2D在频域进行全局特征混合
3. **特征变换**：通过MLP进行非线性变换
4. **保持信息流**：通过跳连接保持梯度流动

### Block的核心优势

- **频域处理**：AFNO2D在频域进行空间混合，计算复杂度为O(NlogN)
- **全局感受野**：能够捕获长距离依赖关系
- **高效训练**：跳连接机制有助于梯度流动
- **特征融合**：特别适合融合不同来源的特征（如局部+全球特征）

简单来说，Block就是一个强大的特征处理单元，它结合了频域滤波和神经网络变换，非常适合处理气象数据这种具有空间相关性的数据。

## `forward_features`返回的特征层级

从代码中可以看到，`self.pretrained_model.forward_features(global_input)`返回的是**最后一层Block的输出特征**。

### `forward_features`方法详解

```python
def forward_features(self, x):
    B = x.shape[0]
    x = self.patch_embed(x)           # 1. 图像patch嵌入
    x = x + self.pos_embed            # 2. 添加位置编码
    x = self.pos_drop(x)              # 3. Dropout
    
    x = x.reshape(B, self.h, self.w, self.embed_dim)  # 4. reshape
    for blk in self.blocks:           # 5. 经过所有Block处理
        x = blk(x)

    return x                           # 6. 返回最后一层Block的输出
```

### 特征提取流程

1. **Patch Embedding**：将输入图像分成patch并嵌入到高维空间
2. **Position Embedding**：添加位置信息
3. **Dropout**：随机丢弃一些特征
4. **Reshape**：reshape为(B, H, W, embed_dim)格式
5. **Blocks Processing**：依次经过所有Block处理
6. **Return**：返回**最后一层Block的输出**

### 在完整模型中的位置

```python
def forward(self, x):
    x = self.forward_features(x)      # 这里调用forward_features
    x = self.head(x)                   # 然后经过head层
    x = rearrange(...)                 # 最后reshape输出
    return x
```

### 返回的特征特点

- **维度**：(B, H, W, embed_dim)，其中embed_dim=768
- **语义层次**：最高层特征，包含最丰富的语义信息
- **空间结构**：保持了原始的空间结构(H, W)
- **信息内容**：融合了所有Block处理后的特征

简单来说，`forward_features`提取的是模型中"编码器"部分的最终输出，是最高层、最抽象的特征表示。
