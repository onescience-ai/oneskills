# 组件依赖分析器

## 适用场景

使用组件知识生成代码前分析组件依赖关系

## 执行步骤

### 1. 初始化

- 待处理队列 = [当前组件文件]
- 已处理集合 = {}
- 依赖图 = {}

### 2. 循环处理（直到队列为空）

- 取队列首个文件
- 若已处理 → 跳过
- 读取文件内容
- 解析依赖（import 语句、组件引用）
- 对每个依赖项：
  - 项目组件 → 添加到队列
  - 记录到依赖图
- 读取组件知识文件
- 标记已处理

### 3. 输出

依赖图 + 组件知识库
