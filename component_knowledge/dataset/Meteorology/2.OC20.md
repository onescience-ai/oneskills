## 一、数据集核心基础信息（代码生成前置知识）

### 1.1 数据集唯一标识

全称：Open Catalyst 2020（OC20）数据集

唯一主键：sid（样本ID，字符串）

数据类型：吸附体-催化表面结构、能量、力与弛豫任务标签

官方数据源：Open Catalyst Project / FAIR Chemistry Docs

数据规模：133M+ DFT calculations，约 460K adsorbate-catalyst relaxations，3 个核心任务。

补充必要项：需记录任务类型（S2EF/IS2RE/IS2RS）、split 版本与 LMDB 文件来源。

### 1.2 数据对象与解析优先级

| 数据对象 | 代码标识 | 解析优先级 | 处理逻辑 |
| --- | --- | --- | --- |
| 结构到能量与力 | S2EF | 高 | 默认优先解析 |
| 初态到弛豫能量 | IS2RE | 高 | 保留初始结构与标量标签 |
| 初态到弛豫结构 | IS2RS | 中 | 保留目标坐标 |
| 测试/验证 split | VAL_TEST | 中 | 只读不写训练标签 |

## 二、数据格式规范与字段映射（必解析字段清单）

### 2.1 主格式

主格式：LMDB（.lmdb）—— 官方分发优先格式。按 key-value 逐样本读取。

| 字段名 | 代码变量名 | 数据类型 | 是否必填 | 缺失处理 |
| --- | --- | --- | --- | --- |
| sid | sid | 字符串 | 是 | 终止解析 |
| pos | positions | 数组 | 是 | 剔除样本 |
| atomic_numbers | atomic_numbers | 数组 | 是 | 剔除样本 |
| y/energy | energy | 浮点数 | 否 | 测试集保留为空 |
| force | forces | 数组 | 否 | 非 S2EF 任务保留为空 |

### 2.2 配套格式

配套格式：ASE Atoms / NumPy 数组。用于结构对象恢复与数值计算。

| 行标识/字段 | 字段含义 | 代码变量名 |
| --- | --- | --- |
| cell | 晶胞矩阵 | cell |
| tags | 原子标签 | tags |

### 2.3 兼容/辅助格式

兼容/辅助格式：pickle / metadata JSON。仅用于 split 清单与任务映射。

## 六、校验规则与异常处理（判断条件+分支逻辑）

### 6.1 必做校验（代码必须实现，顺序执行）

sid 非空且唯一

positions 与 atomic_numbers 长度一致

forces 维度应为 [n_atoms,3]

能量标签必须为有限值或显式为空

### 6.2 异常处理分支（固定捕获+处理逻辑）

| 异常类型 | 触发场景 | 代码处理逻辑 |
| --- | --- | --- |
| lmdb.Error | 数据库打开失败 | 记录并跳过当前库 |
| EOFError | 样本反序列化失败 | 跳过当前样本 |
| KeyError | 字段缺失 | 按任务类型填空或跳过 |
| MemoryError | 批量读取过大 | 缩小 batch size 并重试一次 |

## 七、数据过滤与清洗规则（剔除条件清单）

过滤 positions 为空的样本

过滤 atomic_numbers 非法样本

过滤重复 sid

过滤标签缺失且不属于测试集的样本
