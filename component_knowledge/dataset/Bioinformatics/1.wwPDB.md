## 二、数据格式规范与字段映射（必解析字段清单）

### 2.1 主格式：PDBx/mmCIF（.cif）—— 代码优先解析

格式类型：键值对文本格式，以data_ 开头，以 # 结尾；代码按类别+关键字提取字段，禁止按行切片解析。

| 类别关键字 | 字段名 | 代码提取变量名 | 数据类型 | 是否必填 | 缺失处理 |
| --- | --- | --- | --- | --- | --- |
| _entry | id | pdb_id | 字符串 | 是 | 终止解析，标记无效ID |
| _exptl | method | expt_method | 字符串 | 是 | 标记为未知方法，继续解析 |
| _refine | ls_d_res_high | resolution | 浮点数 | 是 | 标记为无分辨率，过滤出局 |
| _atom_site | Cartn_x, Cartn_y, Cartn_z | coord | 数组[float,float,float] | 是 | 剔除当前原子，保留其他原子 |
| _atom_site | B_iso_or_equiv | bfactor | 浮点数 | 否 | 赋值为0.00 |
| _atom_site | occupancy | occupancy | 浮点数 | 否 | 赋值为1.00 |
| _struct_asym | id | chain_id | 字符串 | 是 | 按顺序自动生成链ID |

### 2.2 兼容格式：PDB（.pdb）

格式类型：固定行宽文本格式，代码按字符起始-结束位置切片提取；仅当CIF格式下载失败时启用。

| 行标识 | 字段含义 | 字符区间 | 代码变量名 |
| --- | --- | --- | --- |
| ATOM | 原子名称 | 12-16 | atom_name |
| ATOM | X坐标 | 30-38 | coord[0] |
| ATOM | Y坐标 | 38-46 | coord[1] |
| ATOM | Z坐标 | 46-54 | coord[2] |
| ATOM | B因子 | 60-66 | bfactor |
| END | 文件结束 | 全行 | 终止解析信号 |
