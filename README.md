OneScience 平台
===============

为科学计算而生，为智能研发而建
------------------------------

OneScience 是一个面向科学计算和智能研发的全栈平台，深度融合先进的深度学习框架与科学计算工具链。平台通过模块化设计、标准化接口和智能体友好架构，大幅降低科学模型开发难度，加速从算法研究到工程落地的全流程。

核心优势
~~~~~~~~

🧬 多领域覆盖，开箱即用的解决方案
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

OneScience 深度支持地球气候科学、生命信息、计算流体、工业仿真、材料化学等多个前沿领域：

* **地球科学**：集成Pangu、XiHe、FuXi等国际认可的气候预报模型，支持气象、海洋、土壤等多参数多尺度模拟
* **生命信息**：提供Protenix、Evo2等蛋白质结构预测与序列学习模型，加速生物信息研究
* **材料化学**：内置MACE等多体势模型，支持分子动力学模拟与材料性质预测
* **计算流体**：包含CFDBenchmark 及相关求解工具，面向工程流动问题求解

每个领域均提供完整的数据集、热点模型和标准化工作流，研究者可快速基于现有能力进行定制和扩展。

🔧 像搭积木一样组织科研开发
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

平台构建了灵活高效的组件生态：

* **数据标准化**：多学科标准数据读取、解析与组织框架，支持NetCDF、HDF5等多种科学数据格式，消除多领域多数据处理问题，提供统一标准

* **基础组件库**：提供100+可复用的核心模块，包括Attention、Transformer、Encoder/Decoder、Embedding、Sampling等多组件模块，以及分布式训练、混合精度计算、梯度累积等科学辅助工具，可用于快速搭建科学模型，也可用于定制化改造

* **热点模型库**：集成复现多个领域的SOTA模型，支持模型架构定制、损失函数变更、推理优化全流程，用户既可快速调用，也可深度改造

* **超算友好设计**：专为超算中心设计，原生支持多卡分布式训练、DCU/GPU 异构计算、高性能通信库集成，充分利用国产与国际先进算力资源

⚡ 智能体原生协同框架
^^^^^^^^^^^^^^^^^^^^^

OneScience 将科学研发流程自动化和智能化，为智能体深度参与研发创造条件：

* **可调用能力封装**：将数据集、组件、工具、模型统一包装成标准化的可调用接口，支持通过API、Prompt等多种方式描述，作为skills供智能体调用

* **自动化流程编排**：智能体可根据研究目标自主完成模块选择、数据流设计、模型配置、超参优化等复杂决策

* **端到端工程闭环**：从需求理解、方案设计、代码生成、模型训练、结果验证等全环节构建闭合反馈，支持迭代优化和快速迭代

🚀 性能与易用的完美结合
^^^^^^^^^^^^^^^^^^^^^^^^^

* **高效的计算引擎**：深度优化的混合精度计算、张量编译、算子融合等技术，在保证精度的前提下显著提升训练与推理效率

* **无缝的超算集成**：内置模块加载器、环境适配器、资源调度接口，支持快速部署到国家超算中心、本地集群等多种计算环境

* **详尽的文档与案例**：提供从环境配置、模型训练、推理应用、Agent 驱动开发等多个维度的最佳实践文档，降低学习成本

为什么选择 OneScience
~~~~~~~~~~~~~~~~~~~~~

✅ **加速创新** — 从零到论文发表只需数周，而非数月

✅ **降低门槛** — 无需深厚的工程背景，也能实现高性能科学计算

✅ **支撑协作** — 智能体与人类研究者无缝协作，提升研发效率

快速开始
~~~~~~~~

安装手册
  OneScience 的快速开始入口，面向 DCU、GPU 等不同平台提供安装、配置与环境验证指南，帮助用户高效完成部署，快速进入开发流程

模块列表
  平台基础组件总览，涵盖 Attention、Transformer、Encoder/Decoder、Embedding、Sample 等核心组件以及分布式训练、数据处理等科学辅助工具，汇聚百余种可复用组件，支撑科学模型的高效开发与灵活组合

热点模型
  平台集成并复现多领域热点模型，覆盖地球科学（Pangu、XiHe、FuXi 等）、生命信息（Protenix、Evo2）、材料化学（MACE）、计算流体（CFD Benchmark）等方向，支持从模型适配到训练推理的全流程开发

开发案例
  基于真实科研场景的实践案例，展示如何依托 OneScience 完成模型复现、模块改造与工程落地，为平台化开发提供参考路径

Agent 驱动开发
  面向智能体协作研发的开发范式，依托平台内置的组件、模型、数据与知识能力，支持智能体围绕需求 prompt 完成任务规划、能力调用、代码生成与工程验证

数据说明
  平台数据能力与数据集接口说明，提供面向多领域科研任务的统一数据读取、解析与组织方式，为模型开发与训练提供标准化支撑
